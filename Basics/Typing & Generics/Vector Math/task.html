Implement Type-Safe version of sum and sub functions which can work with all type of Vectors but ensures at compile-time that only Vectors of the same Type (hence same length) can be summed.

For example:
  • The following should compile successfully 
          
val result: Vector3 = sum(Vector3(1, 2, 3), Vector3(3, 2, 1))
  • The following should fail at the compile time 
          
val result: ??? = sum(Vector2(1, 2, 3), Vector3(3, 2, 1)) // no way to guess result Vector type
